## 1. Построение порождающего многочлена

Для БЧХ-кода (15,7) с минимальным расстоянием 5 необходимо, чтобы порождающий многочлен $g(x)$ имел корнями $\alpha^1,\alpha^2,\alpha^3,\alpha^4$ в поле $\mathrm{GF}(2^4)$.

Минимальные многочлены для циклотомических классов:

- Для $\alpha,\alpha^2,\alpha^4,\alpha^8$: $m_1(x)=x^4+x+1$
- Для $\alpha^3,\alpha^6,\alpha^{12},\alpha^9$: $m_3(x)=x^4+x^3+x^2+x+1$

Порождающий многочлен:

$$g(x)=m_1(x)\cdot m_3(x)=(x^4+x+1)(x^4+x^3+x^2+x+1)$$

Раскрываем скобки в $\mathrm{GF}(2)$:

$$
\begin{aligned}
&(x^4+x+1)(x^4+x^3+x^2+x+1)=\\
&=x^8+x^7+x^6+x^5+x^4\\
&\quad+x^5+x^4+x^3+x^2+x\\
&\quad+x^4+x^3+x^2+x+1
\end{aligned}
$$

Суммируем коэффициенты по модулю 2:

| Степень | Коэффициент (mod 2) |
|---------|---------------------|
| $x^8$   | 1 |
| $x^7$   | 1 |
| $x^6$   | 1 |
| $x^5$   | 0 (1+1) |
| $x^4$   | 1 (1+1+1) |
| $x^3$   | 0 (1+1) |
| $x^2$   | 0 (1+1) |
| $x^1$   | 0 (1+1) |
| $x^0$   | 1 |

Итак,

$$\boxed{g(x)=x^8+x^7+x^6+x^4+1}$$

Вектор коэффициентов (от $x^0$ до $x^8$): $\mathbf{g}=[1,0,0,0,1,0,1,1,1]$.

## 2. Порождающая матрица

### 2.1. Циклическая форма

Строки матрицы — циклические сдвиги $g(x)$ (коэффициенты от $x^0$ до $x^{14}$):

### 2.2. Систематический вид

Находим остатки $x^{k-1+i}\bmod g(x)$ для $i=0,\dots,6$ ($k=7$, $n-k=8$).  
Из $g(x)=x^8+x^7+x^6+x^4+1$ следует:

$$x^8 \equiv x^7+x^6+x^4+1 \pmod{g(x)}$$

Вычисляем степени (коэффициенты при $x^7\ldots x^0$):

| $j$ | $x^j \bmod g(x)$ |
|-----|------------------|
| 7   | 00000001 |
| 8   | 11010001 |
| 9   | 11110010 |
|10   | 11100110 |
|11   | 11001110 |
|12   | 10011110 |
|13   | 00111111 |

Систематическая порождающая матрица (первые 7 столбцов — единичная, последние 8 — проверочные):

$$
\boxed{G=
\begin{pmatrix}

## 3. Декодирование принятой последовательности

Принято: $\mathbf{r}=111001100011000$.  
Биты нумеруются слева направо от $x^{14}$ до $x^0$.

Полиномиальная форма:

$$r(x)=x^{14}+x^{13}+x^{12}+x^9+x^8+x^4+x^3$$

### 3.1. Вычисление синдрома делением на $g(x)$

Делим $r(x)$ на $g(x)=x^8+x^7+x^6+x^4+1$:

1. $x^{14}$ даёт частное $x^6$:  
   $x^6\cdot g(x)=x^{14}+x^{13}+x^{12}+x^{10}+x^6$  
   Остаток: $r_1(x)=x^{10}+x^9+x^8+x^6+x^4+x^3$

2. $x^{10}$ даёт частное $x^2$:  
   $x^2\cdot g(x)=x^{10}+x^9+x^8+x^6+x^2$  
   Остаток: $r_2(x)=x^4+x^3+x^2$

Степень остатка меньше 8, поэтому

$$s(x)=r(x)\bmod g(x)=x^4+x^3+x^2\neq0$$

Синдром ненулевой — в принятом слове есть ошибки.

### 3.2. Вычисление компонент синдрома в поле $\mathrm{GF}(2^4)$

Используем неприводимый многочлен $x^4+x+1$. Таблица степеней $\alpha$:

| Степень | Вектор $(x^3x^2x^1x^0)$ | Степень | Вектор |
|---------|------------------------|---------|------------------------|
| $\alpha^0$ | 0001 | $\alpha^8$ | 0101 |
| $\alpha^1$ | 0010 | $\alpha^9$ | 1010 |
| $\alpha^2$ | 0100 | $\alpha^{10}$ | 0111 |
| $\alpha^3$ | 1000 | $\alpha^{11}$ | 1110 |
| $\alpha^4$ | 0011 | $\alpha^{12}$ | 1111 |
| $\alpha^5$ | 0110 | $\alpha^{13}$ | 1101 |
| $\alpha^6$ | 1100 | $\alpha^{14}$ | 1001 |
| $\alpha^7$ | 1011 | | |

**Синдром $S_1 = r(\alpha)$**:

$$S_1 = \alpha^{14}+\alpha^{13}+\alpha^{12}+\alpha^9+\alpha^8+\alpha^4+\alpha^3$$

Подставляем векторы и выполняем XOR:

$$
\begin{aligned}
&1001 \oplus 1101 = 0100,\\
&0100 \oplus 1111 = 1011,\\
&1011 \oplus 1010 = 0001,\\
&0001 \oplus 0101 = 0100,\\
&0100 \oplus 0011 = 0111,\\
&0111 \oplus 1000 = 1111.
\end{aligned}
$$

Итак, $S_1 = 1111 = \alpha^{12}$.

**Синдром $S_2$** (используем $S_2=S_1^2$):

$$S_2 = (\alpha^{12})^2 = \alpha^{24} = \alpha^{24\bmod15}= \alpha^9 \quad (\text{вектор }1010)$$

**Синдром $S_3 = r(\alpha^3)$**:

$$
\begin{aligned}
S_3 &= \alpha^{42}+\alpha^{39}+\alpha^{36}+\alpha^{27}+\alpha^{24}+\alpha^{12}+\alpha^9 \\
&= \alpha^{12}+\alpha^9+\alpha^6+\alpha^{12}+\alpha^9+\alpha^{12}+\alpha^9 \\
&= \alpha^{12}+\alpha^9+\alpha^6.
\end{aligned}
$$

Векторы: $\alpha^{12}=1111$, $\alpha^9=1010$, $\alpha^6=1100$:

$$1111\oplus1010=0101,\quad 0101\oplus1100=1001$$

Получаем $S_3 = 1001 = \alpha^{14}$.

**Синдром $S_4 = S_2^2$**:

$$S_4 = (\alpha^9)^2 = \alpha^{18} = \alpha^{3} \quad (\text{вектор }1000)$$

Сводка:

| $i$ | $S_i$ | Двоичный вектор |
|-----|-------|----------------|
| 1   | $\alpha^{12}$ | 1111 |
| 2   | $\alpha^{9}$  | 1010 |
| 3   | $\alpha^{14}$ | 1001 |
| 4   | $\alpha^{3}$  | 1000 |

### 3.3. Поиск локаторного многочлена ошибок

Предполагаем $\nu=2$ ошибки. Уравнения для коэффициентов $\sigma_1,\sigma_2$:

$$
\begin{cases}
S_1\sigma_1 + S_2\sigma_2 = S_3,\\
S_2\sigma_1 + S_3\sigma_2 = S_4.
\end{cases}
$$

Определитель системы:

$$\Delta = S_1S_3 + S_2^2 = \alpha^{12}\alpha^{14} + \alpha^{18} = \alpha^{26} + \alpha^{18} = \alpha^{11} + \alpha^3$$

$\alpha^{11}=1110$, $\alpha^3=1000$ → $1110\oplus1000=0110=\alpha^5$. Итак, $\Delta=\alpha^5$.

Находим $\sigma_2$:

$$\sigma_2 = \frac{S_3^2 + S_2S_4}{\Delta} = \frac{\alpha^{28}+\alpha^{12}}{\alpha^5}$$

$\alpha^{28}=\alpha^{13}$ ($28\bmod15=13$), $\alpha^{13}\oplus\alpha^{12}=1101\oplus1111=0010=\alpha^1$.  
Тогда $\sigma_2 = \alpha^1/\alpha^5 = \alpha^{-4}= \alpha^{11}$.

Находим $\sigma_1$:

$$\sigma_1 = \frac{S_1S_4 + S_2S_3}{\Delta} = \frac{\alpha^{15}+\alpha^{23}}{\alpha^5}$$

$\alpha^{15}=1$, $\alpha^{23}=\alpha^{8}$ ($23\bmod15=8$). $1\oplus\alpha^8 = 0001\oplus0101=0100=\alpha^2$.  
$\sigma_1 = \alpha^2/\alpha^5 = \alpha^{-3}= \alpha^{12}$.

Локаторный многочлен:

$$\boxed{\sigma(x)=1 + \alpha^{12}x + \alpha^{11}x^2}$$

### 3.4. Определение позиций ошибок

Ищем $i\in\{0,\dots,14\}$ такие, что $\sigma(\alpha^{-i})=0$, т.е. $1 + \alpha^{12-i} + \alpha^{11-2i}=0$.

Результаты вычислений (векторы):

| $i$ | $\alpha^{12-i}$ | $\alpha^{11-2i}$ | $1\oplus\alpha^{12-i}\oplus\alpha^{11-2i}$ |
|-----|----------------|------------------|---------------------------------------------|
| 0   | $\alpha^{12}=1111$ | $\alpha^{11}=1110$ | $0000$ |
| 1   | $\alpha^{11}=1110$ | $\alpha^{9}=1010$ | $0101$ |
| 2   | $\alpha^{10}=0111$ | $\alpha^{7}=1011$ | $1101$ |
| 3   | $\alpha^{9}=1010$ | $\alpha^{5}=0110$ | $1101$ |
| 4   | $\alpha^{8}=0101$ | $\alpha^{3}=1000$ | $1100$ |
| 5   | $\alpha^{7}=1011$ | $\alpha^{1}=0010$ | $1000$ |
| 6   | $\alpha^{6}=1100$ | $\alpha^{14}=1001$ | $0100$ |
| 7   | $\alpha^{5}=0110$ | $\alpha^{12}=1111$ | $1000$ |
| 8   | $\alpha^{4}=0011$ | $\alpha^{10}=0111$ | $0101$ |
| 9   | $\alpha^{3}=1000$ | $\alpha^{8}=0101$ | $1100$ |
| 10  | $\alpha^{2}=0100$ | $\alpha^{6}=1100$ | $1001$ |
| 11  | $\alpha^{1}=0010$ | $\alpha^{4}=0011$ | $0000$ |

Нули получены при $i=0$ и $i=11$. Это позиции ошибок (отсчёт от $x^0$).

Принятая последовательность $\mathbf{r}$ (биты $x^{14}\dots x^0$):

$$r_{14}=1,\; r_{13}=1,\; r_{12}=1,\; r_{11}=0,\; r_{10}=0,\; r_{9}=1,\; r_{8}=1,\; r_{7}=0,\; r_{6}=0,\; r_{5}=0,\; r_{4}=1,\; r_{3}=1,\; r_{2}=0,\; r_{1}=0,\; r_{0}=0.$$

Инвертируем биты в позициях $i=11$ ($x^{11}$) и $i=0$ ($x^0$):

- $r_{11}=0$ → $1$
- $r_{0}=0$ → $1$

Исправленное кодовое слово:

$$\boxed{\mathbf{c}=1\;1\;1\;1\;0\;1\;1\;0\;0\;0\;1\;1\;0\;0\;1}$$

В полиномиальном виде:

$$c(x)=x^{14}+x^{13}+x^{12}+x^{11}+x^9+x^8+x^4+x^3+1$$

Проверка: $c(x)$ делится на $g(x)$ (остаток от деления равен нулю), что подтверждает корректность декодирования.
